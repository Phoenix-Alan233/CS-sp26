# 信息理论 2 小时速通计划

你现在有概率论基础，但信息理论刚开始学。别把它想成一门全新的玄学：它主要是在问三件事：

1. 一个随机东西“不确定”到什么程度？这叫熵。
2. 看到另一个随机东西后，不确定性减少了多少？这叫互信息。
3. 如果要把信源输出写成 0/1 串，平均最少要多少 bit？这叫信源编码。

## 1. Lecture1 用人话梳理：信息的度量

### 1.1 自信息：越罕见越“有信息”

一个事件 `X=x` 的自信息：

```text
I(x) = -log p(x)
```

概率越小，自信息越大；概率为 1，自信息为 0。

注意：单个事件之间的互信息 `I(x;y)=log[p(x,y)/(p(x)p(y))]` 可以小于 0，因为某个 `y` 的出现可能让某个 `x` 更不可能。但随机变量互信息 `I(X;Y)` 是平均后的结果，永远非负。

### 1.2 熵：平均自信息，也就是平均不确定性

离散随机变量：

```text
H(X) = - sum_x p(x) log p(x)
```

常用性质：

```text
0 <= H(X) <= log |X|
H(X)=0  <=> X 完全确定
H(X)=log |X| <=> X 在 |X| 个值上均匀分布
```

二元熵函数：

```text
H_b(p) = -p log p - (1-p) log(1-p)
```

它在 `p=1/2` 时最大，最大值 1 bit。

### 1.3 条件熵：知道 Y 后，X 还剩多少不确定

```text
H(X|Y) = - sum_{x,y} p(x,y) log p(x|y)
```

最常用结论：

```text
H(X|Y) <= H(X)
如果 X 与 Y 独立，H(X|Y)=H(X)
如果 X 是 Y 的确定性函数，H(X|Y)=0
```

### 1.4 联合熵：两件事一起的不确定性

链式公式是证明题的核心：

```text
H(X,Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)
H(X,Y,Z) = H(X) + H(Y|X) + H(Z|X,Y)
```

看到证明题，第一反应就是把两边都写成联合熵，或者给联合熵换两种展开方式。

### 1.5 互信息：知道 Y 后，X 的不确定性减少了多少

```text
I(X;Y) = H(X) - H(X|Y)
       = H(Y) - H(Y|X)
       = H(X) + H(Y) - H(X,Y)
```

性质：

```text
I(X;Y) >= 0
I(X;Y) = I(Y;X)
I(X;Y) <= H(X), I(X;Y) <= H(Y)
X 与 Y 独立 <=> I(X;Y)=0
```

单个事件互信息可负；随机变量互信息不可负。这是判断题高频坑。

### 1.6 确定性函数：函数不会凭空增加信息

如果 `Y=f(X)`，那么 `Y` 被 `X` 完全决定：

```text
H(Y|X)=0
H(X,Y)=H(X)
H(Y) <= H(X)
```

如果 `f` 还是一对一函数，则 `Y` 和 `X` 可互相确定：

```text
H(Y)=H(X)
H(Y|X)=0
H(X|Y)=0
H(X,Y)=H(X)=H(Y)
I(X;Y)=H(X)=H(Y)
```

这张卷子的第 4、6 题就是这个模板。

### 1.7 马尔可夫链：信息处理不等式

如果 `X -> Y -> Z`，意思是 `Z` 只能通过 `Y` 间接知道 `X`，后处理不会增加关于 `X` 的信息：

```text
I(X;Y) >= I(X;Z)
I(Y;Z) >= I(X;Z)
```

试卷判断题第 1 题第 (3) 就考这个。

### 1.8 连续随机变量：微分熵

连续型随机变量：

```text
h(X) = - integral f(x) log f(x) dx
```

它和离散熵很像，但有两个大坑：

1. 微分熵可以为负。
2. 微分熵不是绝对的不确定性尺度，变量缩放会改变它。

变量缩放公式：

```text
Y = aX, a != 0
h(Y) = h(X) + log |a|
```

高斯最大熵：

```text
方差固定时，高斯分布的微分熵最大。
```

指数分布 `f(x)=e^{-x}, x>=0`：

```text
h(X) = - integral_0^infty e^{-x} log(e^{-x}) dx
     = integral_0^infty x e^{-x} dx
     = 1 nat
```

如果考试用 bit，则 `1 nat = log_2 e bit`。但第 5 题通常按自然对数算，答案 `h(X)=1`。

## 2. Lecture2 用人话梳理：无损压缩

### 2.1 信源编码在问什么

信源输出符号，例如 `a1,a2,...`，每个符号有概率。压缩就是给常见符号短码字，给罕见符号长码字，让平均码长最小。

平均码长：

```text
n_bar = sum_i p_i n_i
```

编码效率：

```text
eta = H(U) / (n_bar log D)
```

二元码 `D=2` 时：

```text
eta = H(U) / n_bar
```

### 2.2 等长编码、典型列、Shannon 信源编码定理

长序列里，真正高概率出现的是典型列。典型列的关键直觉：

```text
每个典型列概率约为 2^{-L H(U)}
典型列个数约为 2^{L H(U)}
典型列总概率趋近 1
```

注意两个坑：

- 典型列集合元素个数不一定比非典型集合多。
- 二元 Bernoulli 信源中，全 0 序列是否典型，要看参数。若 `p(0)=1`，全 0 就是典型；若 `0<p(0)<1` 且不是退化分布，全 0 通常不是典型。

Shannon 信源编码定理：

```text
长块编码时，平均每符号码率可以逼近 H(U)。
```

### 2.3 唯一可译码、即时码、Kraft 不等式

即时码/前缀码：没有任何码字是另一个码字的前缀。即时码一定唯一可译。

唯一可译码：码字串拼起来后能唯一拆回原消息。它不一定是即时码。

Kraft 不等式：

```text
存在 D 元前缀码长度 n1,...,nK 的充要条件：
sum_i D^{-n_i} <= 1

任何 D 元唯一可译码也必须满足：
sum_i D^{-n_i} <= 1
```

所以判断“不存在某组码长的唯一可译码”，先算 Kraft 和：

```text
sum_i D^{-n_i} > 1  => 一定不存在唯一可译码
sum_i D^{-n_i} <= 1 => 至少存在同长度前缀码，因此存在唯一可译码
```

例如二元码长 `{1,2,3,3}`：

```text
2^{-1}+2^{-2}+2^{-3}+2^{-3}=1
```

所以存在唯一可译码。判断题说“不存在”就是错。

### 2.4 Huffman 编码：平均码长最短的不等长码

二元 Huffman：

1. 每次取概率最小的两个合并。
2. 合并后的概率再放回去排序。
3. 从根到叶标 0/1 得码字。

重要性质：

```text
概率越小，码长越长或不短。
二元 Huffman 中，概率最小的两个符号码长相等，且最后一位不同。
D 元 Huffman 中，每次一般合并概率最小的 D 个节点；若需要补虚符号，先补 0 概率虚符号。
因此实际信源中概率最小的两个符号仍会在同一次合并中成为兄弟节点，码长相等。
```

D 元 Huffman 还要补虚符号，使：

```text
(K + s - 1) mod (D - 1) = 0
```

其中 `s` 是补的 0 概率虚符号数。

### 2.5 Shannon 编码

Shannon 编码给每个符号码长：

```text
l_i = ceil(-log p_i)
```

它一定是前缀码，并满足：

```text
H(U) <= n_bar < H(U)+1
```

Huffman 是给定分布下最优的前缀码/唯一可译码，因此：

```text
n_bar(Huffman) <= n_bar(Shannon)
```

注意判断题里的“必然大于”：可能相等，所以“必然大于 Huffman”通常是错，应该说“不小于”。

## 3. 试卷.pdf 每题考点与秒杀入口

### 一、判断题：10 个小坑

1. 随机事件互信息可小于 0，随机变量互信息也可小于 0。  
   考点：事件互信息 vs 随机变量互信息。入口：`I(x;y)` 可负，`I(X;Y)>=0`。结论：后半句错。

2. 连续随机变量微分熵越大，说明不确定性越大。  
   考点：微分熵解释。入口：微分熵会随坐标缩放变化，也可为负；不能简单当绝对不确定性。结论：通常判错。

3. `X -> Y -> Z` 则 `I(X;Y) >= I(X;Z)`。  
   考点：数据处理不等式。入口：后处理不增加关于 `X` 的信息。结论：对。

4. 对任何二元 Bernoulli 信源，全零序列都不是典型列。  
   考点：典型列。入口：退化 Bernoulli `p(0)=1` 时全 0 是典型；“任何”太绝对。结论：错。

5. Shannon 编码平均码长必然大于 Huffman 平均码长。  
   考点：Shannon vs Huffman。入口：Huffman 最优，所以 Shannon 不小于 Huffman，但可能相等。结论：错。

6. 不存在码长 `{1,2,3,3}` 的唯一可译码。  
   考点：Kraft 不等式。入口：`1/2+1/4+1/8+1/8=1`，存在前缀码，因此存在唯一可译码。结论：错。

7. 平稳信源平均每符号熵 `H_N(X)` 随 `N` 增大单调递减。  
   考点：平稳信源熵率。入口：Lecture1 结论：`H_N` 非增，趋向熵率。结论：对。

8. D 元 Huffman 中，概率最小的两个符号码长相等。  
   考点：D 元 Huffman。入口：D 元每次合并最小的 D 个节点，必要时先补 0 概率虚符号；实际概率最小的两个符号会同次合并，码长相等。结论：对。

9. 方差一定时，高斯连续变量微分熵最大。  
   考点：最大熵定理。结论：对。

10. 典型列集合元素个数必然大于非典型集合。  
    考点：典型列个数 vs 概率质量。入口：典型集是高概率集，但数量不一定更多。结论：错。

### 二、`Z=X+Y` 证明题

考点：条件熵、确定性变换、独立性。

入口：

```text
给定 Z 后，X 与 Y 一一对应：Y=Z-X，X=Z-Y
所以 H(X|Z)=H(Y|Z)
```

第二问若 `X,Y` 独立：

```text
H(Z|X)=H(X+Y|X)=H(Y|X)=H(Y)
I(X;Z)=H(Z)-H(Z|X)=H(Z)-H(Y)
又 I(X;Z)<=H(X)
```

更直接：

```text
H(Z|Y)=H(X+Y|Y)=H(X|Y)=H(X)
H(Z) >= H(Z|Y)=H(X)
```

### 三、Huffman 编码与效率

考点：Lecture2 Huffman 最佳二元编码、平均码长、编码效率。

信源概率：

```text
0.30, 0.25, 0.15, 0.15, 0.10, 0.05
```

一种二元 Huffman 合并过程：

```text
0.05+0.10=0.15
0.15+0.15=0.30
0.15+0.25=0.40
0.30+0.30=0.60
0.40+0.60=1
```

可得到一组码长：

```text
a1: 2
a2: 2
a3: 2
a4: 3
a5: 4
a6: 4
```

对应的一组可用码字，例如：

```text
a1: 10
a2: 01
a3: 00
a4: 110
a5: 1111
a6: 1110
```

只要码长结构和前缀性对，Huffman 码字不唯一。

平均码长：

```text
n_bar = 0.3*2 + 0.25*2 + 0.15*2 + 0.15*3 + 0.1*4 + 0.05*4
      = 2.45 bits/symbol
```

熵：

```text
H(U) = -sum p_i log p_i = 2.3905 bits
```

效率：

```text
eta = H(U)/n_bar = 2.3905/2.45 = 0.9757 ≈ 97.57%
```

Huffman 码不唯一，码字可以不同，但码长和平均码长应一致。模拟卷答案给过类似结果约 `97.55%`。

### 四、确定性函数 `Y=g(X), Z=h(Y)`

考点：函数不会增加熵、链式公式。

第一问：

```text
Y=g(X) => H(Y|X)=0
H(X,Y)=H(X)+H(Y|X)=H(X)
H(X,Y)=H(Y)+H(X|Y)>=H(Y)
所以 H(X)>=H(Y)
```

第二问：

```text
H(Y|Z) <= H(Y) <= H(X)
```

或者用 `Z=h(Y)` 表示 `Z` 是 `Y` 的函数，知道 `Z` 后 `Y` 的剩余不确定性不超过 `Y` 本身。

### 五、指数分布微分熵与缩放

考点：连续熵定义、变量缩放。

```text
f(x)=e^{-x}, x>=0
h(X)=- integral_0^infty e^{-x} log(e^{-x}) dx
    = integral_0^infty x e^{-x} dx
    = 1
```

若 `Y=aX`：

```text
h(Y)=h(X)+log|a|=1+log|a|
```

令 `h(Y)=0`：

```text
1+log|a|=0
|a|=e^{-1}
a=±e^{-1}
```

如果全卷明确用 bit，则把 `h(X)=log_2 e`，方程 `log_2 e + log_2|a|=0`，仍然得到 `|a|=e^{-1}`。

### 六、一对一函数与一般函数

考点：确定性函数、互信息。

已知 `H(X)=4 bits`，`Y=f(X)` 一对一：

```text
H(Y)=4
H(Y|X)=0
H(X|Y)=0
H(X,Y)=4
I(X;Y)=4
```

若 `Z=g(X)` 一般函数：

```text
H(Z|X)=0
H(X,Z)=H(X)=4
H(X,Z)=H(Z)+H(X|Z)>=H(Z)
所以 H(Z)<=4
```

### 七、NBA 七场四胜制

考点：离散随机变量分布、熵、互信息。

第 (1) 问：比赛场次 `X` 取值：

```text
X in {4,5,6,7}
```

恰好第 `k` 场结束，冠军队第 `k` 场赢，前 `k-1` 场里冠军队赢 3 场。两队对称：

```text
P(X=k)=2 * C(k-1,3) / 2^k, k=4,5,6,7
```

所以：

```text
P(X=4)=1/8
P(X=5)=1/4
P(X=6)=5/16
P(X=7)=5/16
```

熵：

```text
H(X)= -sum p log p ≈ 1.9238 bits
```

第 (2) 问：`Y` 是第一场赢家，`Z` 是总冠军。二者都是均匀二元变量，所以：

```text
I(Y;Z)=H(Z)-H(Z|Y)
```

给定第一场 A 赢，A 最终夺冠概率：

```text
P(Z=A | Y=A)
= sum_{t=3}^6 C(t-1,2)/2^t
= 21/32
```

因此给定 `Y` 后，`Z` 是二元分布 `(21/32, 11/32)`：

```text
I(Y;Z) = 1 - H_b(21/32)
       ≈ 0.0716 bits
```

### 八、`Y=XZ` 混合变量

考点：独立变量、函数变换、熵最大化。

题设：

```text
Z in {0,1,-1}
P(Z=0)=1-p
P(Z=1)=P(Z=-1)=p/2
X in {1,2,...,n}, P(X=i)=q_i
X 与 Z 独立
Y=XZ
```

`Y` 的取值是：

```text
0, ±1, ±2, ..., ±n
```

分布：

```text
P(Y=0)=1-p
P(Y=i)=p q_i/2
P(Y=-i)=p q_i/2
```

所以：

```text
H(Y)
= H_b(p) + p * [1 + H(X)]
= H(Z) + p H(X)
```

因为：

```text
H(Z)=H_b(p)+p
```

最大化时，先令 `X` 均匀：

```text
q_i=1/n
H(X)=log n
```

于是：

```text
H(Y)=H_b(p)+p(1+log n)=H_b(p)+p log(2n)
```

求导：

```text
d/dp = log((1-p)/p) + log(2n)
```

令导数为 0：

```text
(1-p)/p * 2n = 1
p = 2n/(2n+1)
```

最大值：

```text
H(Y)_max = log(2n+1)
```

直觉：最大熵就是让 `Y` 在 `2n+1` 个取值 `{0,±1,...,±n}` 上均匀分布，所以 `P(Y=0)=1/(2n+1)`，每个非零点概率也是 `1/(2n+1)`。

```text
H(X)=-sum p log p
H(X,Y)=H(X)+H(Y|X)=H(Y)+H(X|Y)
I(X;Y)=H(X)-H(X|Y)=H(Y)-H(Y|X)=H(X)+H(Y)-H(X,Y)
I(X;Y)>=0；单个事件互信息可负

Huffman: 每次合并最小概率；二元时最小两个码长相等

```
