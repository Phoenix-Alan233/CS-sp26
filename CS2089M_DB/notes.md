# 数据库系统笔记

## 知识框架

- 数据库系统概述：DBMS、数据抽象、数据模型、数据库语言、数据库体系结构。
- 关系模型：关系、元组、属性、候选键、主键、外键、完整性约束、关系代数。
- SQL：DDL、DML、连接、嵌套查询、聚合、视图、触发器、函数、事务语句。
- E-R 模型：实体集、联系集、属性、弱实体、基数约束、参与约束、E-R 到关系模式转换。
- 关系数据库设计：函数依赖、闭包、范式、无损分解、依赖保持。
- 存储与索引：文件组织、页、记录、缓冲区、B+ 树、散列索引。
- 查询处理与优化：选择、连接、排序、代价估计、执行计划、等价变换。
- 事务：ACID、调度、可串行化、锁、死锁、日志、恢复。

## 复习重点

- SQL 题：先确认输出粒度，再写 `FROM/JOIN`，最后处理 `WHERE/GROUP BY/HAVING/ORDER BY`。
- 设计题：先列实体和联系，再补主键、外键、基数和完整性约束。
- 范式题：先求属性闭包和候选键，再判断 2NF、3NF、BCNF。
- 索引题：区分 B+ 树、静态散列、动态散列的查询和更新代价。
- 事务题：按冲突边建图判断可串行化，按锁协议判断是否满足隔离要求。
